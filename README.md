About Todo Application 

A simple, interactive command-line to-do list application written in Python. This project uses Python's built-in `json` and `os` libraries to automatically save your tasks so you never lose your progress!

# Features 

1. Add Task : Adds the Tasks in the json file 
2. View Task : Shows the Tasks present in the json file 
3. Automatically saves the datain the json file
